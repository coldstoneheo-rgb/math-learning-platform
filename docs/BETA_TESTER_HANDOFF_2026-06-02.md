# 베타 테스터 핸드오프 - 2026-06-02

이 문서는 1차 베타 참가자(교사 1명, 학부모 2명, 학생 2명)가 처음 접속했을 때 무엇을 확인하면 되는지와, 문제가 생겼을 때 어떤 정보를 전달하면 되는지 정리한 운영용 안내서입니다.

## 베타 목적

이번 베타의 핵심 질문은 기능 수가 아니라 다음 세 가지입니다.

1. 로그인 후 각 역할의 첫 화면에 안정적으로 도달하는가?
2. 학생의 현재 학습 상태와 성장 흐름을 이해할 수 있는가?
3. 문제가 생겼을 때 교사가 빠르게 원인을 파악하고 복구할 수 있는가?

## 교사 체크

처음 접속 후 아래 순서로 확인합니다.

1. 로그인 후 교사 대시보드에 진입합니다.
2. 학생 관리에서 베타 학생 2명의 이름과 연결 상태를 확인합니다.
3. 리포트 목록에서 각 학생의 최근 리포트가 열리는지 확인합니다.
4. 과거 학습자료 가져오기에서 CSV/이미지 업로드 화면이 보이는지 확인합니다.
5. AI 기억 서랍 관리에서 학생별 과거 기억 준비 상태를 확인합니다.

정상 빈 상태는 오류가 아닙니다. 아직 리포트나 과거 기억이 없는 학생은 “데이터가 부족하다”는 안내가 보여야 하며, 흰 화면이나 무한 로딩이면 오류로 기록합니다.

## 학부모 체크

처음 접속 후 아래 순서로 확인합니다.

1. 로그인 후 학부모 대시보드에 진입합니다.
2. 연결된 자녀 이름이 보이는지 확인합니다.
3. 총 리포트, 성장 여정, 리포트 목록이 보이는지 확인합니다.
4. 리포트 상세 화면을 열어 성장 판단, 핵심 분석, 그래프가 표시되는지 확인합니다.
5. 모바일 화면에서도 주요 내용이 가려지지 않는지 확인합니다.

연결된 자녀가 없는 경우에는 “선생님에게 자녀 연결을 요청해주세요”처럼 다음 행동을 알 수 있는 안내가 보이면 정상입니다.

## 학생 체크

처음 접속 후 아래 순서로 확인합니다.

1. 로그인 후 학생 대시보드에 진입합니다.
2. 연결된 학생 정보가 있으면 내 성장 방향, 총 리포트, 내 리포트 목록을 확인합니다.
3. 아직 연결되지 않았다면 학생 계정 연결 안내와 연결 코드 입력창이 보이는지 확인합니다.
4. 모바일 화면에서 가로 스크롤 없이 핵심 화면이 보이는지 확인합니다.

## 정상 빈 상태 기준

아래 상태는 오류가 아니라 베타 데이터가 아직 준비되지 않았을 때의 정상 화면입니다.

| 역할 | 정상 빈 상태 | 다음 행동 |
| --- | --- | --- |
| 교사 | 오늘 수업이나 최근 리포트 이벤트가 없음 | 학생 연결, 첫 리포트 생성, 과거 학습자료 등록 순서로 확인 |
| 학부모 | 연결된 자녀가 없음 | 학부모 이메일과 자녀 이름을 선생님에게 전달해 연결 요청 |
| 학부모 | 자녀는 보이지만 리포트가 없음 | 선생님이 첫 시험 분석 또는 레벨 테스트 리포트를 생성하면 자동 표시 |
| 학생 | 학생 프로필이 연결되지 않음 | 선생님에게 학생 계정 이메일 연결을 요청하거나 연결 코드 입력 |
| 학생 | 학습 계획이나 리포트가 없음 | 선생님이 분석 리포트를 저장하면 자동 표시 |

반대로 흰 화면, 무한 로딩, 로그인 후 엉뚱한 역할 화면으로 이동, 버튼 클릭 후 아무 반응 없음은 오류로 기록합니다.

## 문제 제보 양식

문제가 생기면 아래 정보를 함께 전달합니다.

- 역할: 교사 / 학부모 / 학생
- 접속한 화면 URL
- 한 줄 증상: 예) 로그인 후 흰 화면, 리포트 클릭 시 오류, 자녀가 보이지 않음
- 발생 시각
- 스크린샷 1장
- 가능하면 사용 기기: PC / 휴대폰 / 태블릿

## 운영자가 보는 지표

베타 중 운영자는 아래 항목을 우선 확인합니다.

- Sentry: 최근 24시간 unresolved issue
- Vercel: 최근 배포 상태와 런타임 에러
- Supabase: 인증/권한 오류, CSV import 실패, 리포트 조회 실패
- 실제 사용자 제보: 로그인 실패, 리포트 접근 실패, 자녀 연결 누락

## 알려진 제한

- 카카오톡 자동 알림은 아직 정식 연동 전입니다. 1차 베타는 수동 안내를 기본으로 합니다.
- 과거 데이터 가져오기는 현재 DB 구조 변경 없이 안전한 범위에서 운영합니다. 상세한 import 이력 테이블은 베타 이후 별도 승인 작업으로 진행합니다.
- AI 기억 서랍은 현재 시험 이미지와 교사 확정값을 대체하지 않습니다. 과거 반복 패턴과 성장 흐름을 보강하는 참고 자료입니다.
